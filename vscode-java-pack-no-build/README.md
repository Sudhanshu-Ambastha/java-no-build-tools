# No-Build Java Power Pack 🛒☕

The definitive extension pack for students, beginners, and developers who want to write Java without the complexity of Maven or Gradle. This suite focuses on a "Zero-Config" philosophy—just open a `.java` file, add your dependencies via JAR Cart, and run.

## 🚀 What's Included?

This pack bundles the essential tools for a high-performance, lightweight Java environment:

- **[JAR Cart](https://marketplace.visualstudio.com/items?itemName=sudhanshuambastha.jar-cart)**: Search and download JAR dependencies directly into your project without build scripts.
- **Language Support for Java™ (Red Hat)**: Smart IntelliSense, error checking, and code navigation.
- **Debugger for Java (Microsoft)**: Lightweight debugging to squash bugs in single-file or multi-file projects.
- **Code Runner**: Adds a one-click "Play" button to run your Java files instantly.
- **Oracle Java Platform**: Streamlined JDK management and additional language features.
- **Test Runner for Java**: Easily run JUnit or TestNG tests.

## 🎯 Why this pack?

Most Java setups force you into the "Enterprise" way of doing things (Maven/Gradle). This pack is built for the **Sovereign Developer** who values speed, simplicity, and logic over boilerplate configuration.
