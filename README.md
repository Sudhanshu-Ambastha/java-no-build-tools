# Java No-Build Tools 🛒☕

This repository hosts multiple VS Code extensions for lightweight Java development without Maven/Gradle.

## 1. JAR Cart 🛒

The **zero-config dependency manager for Java**. Search, select, and "checkout" JAR files directly into your project's `lib` folder.

👉 [Marketplace](https://marketplace.visualstudio.com/items?itemName=sudhanshuambastha.jar-cart-sa)  
👉 [README](./jar-cart/README.md)

## 2. No-Build Java Power Pack ☕

An extension pack bundling JAR Cart with essential Java tools (Red Hat Java, Debugger, Test Runner, Code Runner, Oracle Java).
